概述

在读书的过程中, 许多内容都是看了之后没有实践, 就逐渐丢掉了. 要避免这样的问题, 一个是多实践, 常实践, 另外一个就是要记录下来便于日后查阅. 因此就有了这个文档集.

在里面包含了看的书相关的内容, 以及源码阅读的相关内容, 目录如下:

  1. 书籍的阅读, 包含一些经典的书籍.
  2. 源码阅读, 从 github 找一些值得看的工程来.
